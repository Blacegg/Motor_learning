【*】 程序简介 
-工程名称：直流有刷减速电机-电流电压读取-MOS管搭建板
-实验平台: 野火STM32 F103 拂晓开发板


【 ！】功能简介：
按下KEY1使能或禁用电机,并进入控制模式,按下KEY2\KEY3\KEY4可以调整
占空比,以到达加减速的效果.

【 ！】实验操作：

接线:
电机驱动板 5V\GND 					<----> 开发板 5V\GND
电机驱动板 PWM1\PWM2 				<----> 开发板 PE9\PE11
电机驱动板 SD 							<----> 开发板 PD15
电机驱动板 电压\电流检测 		<----> 开发板 PC1\PC0

注意：使用跳冒连接 J33 和 J34

下载本程序，复位开发板即可。

【 ！！】注意事项：
无

/*********************************************************************************************/

【*】 引脚分配

定时器TIM1输出PWM信号:
两路PWM输出到PE9\PE11上连接到驱动板上的PWM信号输入.
电机驱动板 PWM1\PWM2 <----> 开发板 PE9\PE11

电机使能引脚:
电机驱动板 SD <----> 开发板 PD15

电压电流采集:
电机驱动板 电压\电流检测 		<----> 开发板 PC1\PC0
/*********************************************************************************************/

【*】 时钟

A.晶振：
-外部高速晶振：25MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB1 = 72MHz
-APB2 = 36MHz 
-APB1 = 36MHz

C.浮点运算单元：
  不使用
