【*】 程序简介 
-工程名称：2个直流有刷减速电机按键控制
-实验平台: 野火STM32 F103 指南者开发板


【 ！】功能简介：
上电即使能电机,并进入控制模式,按下KEY1\KEY2 和 KEY3\KEY4可以调整电机1和电机2的占空比,以到达加减速的效果.
    按下KEY1    增大电机1 PWM占空比，电机1加速
    按下KEY2    减小电机1 PWM占空比，电机1减速
    按下KEY3    增大电机2 PWM占空比，电机2加速
    按下KEY4    减小电机2 PWM占空比，电机2减速


【 ！】实验操作：

电机1接线:
电机驱动板 5V\GND 		<----> 开发板 5V\GND
电机驱动板 PWM1\PWM2 	<----> 开发板 PE9\PE11
电机驱动板 SD 				<----> 开发板 PD15

电机2接线:
电机驱动板 5V\GND 		<----> 开发板 5V\GND
电机驱动板 PWM1\PWM2 	<----> 开发板 PC6\PC7
电机驱动板 SD 				<----> 开发板 PC9

下载本程序，复位开发板即可。

【 ！！】注意事项：
无

【*】 引脚分配

定时器TIM1输出PWM信号:
两路PWM输出到PE9\PE11上,连接到驱动板上的PWM信号输入.
电机驱动板 PWM1\PWM2 <----> 开发板 PE9\PE11

电机使能引脚:
电机驱动板 SD <----> 开发板 PB12

定时器TIM3输出PWM信号:
两路PWM输出到PC6\PC7上,连接到驱动板上的PWM信号输入.
电机驱动板 PWM1\PWM2 <----> 开发板 PC6\PC7

电机使能引脚:
电机驱动板 SD <----> 开发板 PB13
